# React Accordion Application

This is a React application built with Vite and TypeScript that demonstrates an interactive accordion component. The accordion allows users to expand and collapse content sections for a better user experience.

## Features

- Interactive accordion with expand/collapse functionality
- Built with React 19 and TypeScript
- Fast development with Vite
- ESLint for code quality

## Getting Started

Follow these steps to run the project locally:

1. Clone the repository to your machine:
   ```bash
   git clone <repository-url>
   cd react-accordion-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173` to view the application.

## Available Scripts

- `npm run dev` - Starts the development server
- `npm run build` - Builds the app for production
- `npm run lint` - Runs ESLint to check for code issues
- `npm run preview` - Previews the production build locally
